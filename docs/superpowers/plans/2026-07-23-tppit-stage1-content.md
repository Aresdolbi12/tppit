# tppit.ru — Этап 1 «Контент»: план реализации

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** наполнить сайт всеми разделами палаты с реальным контентом (источник — исследование основного сайта) и задеплоить на GitHub Pages.

**Architecture:** статичный контент в типизированных ts-модулях `web/src/data/*`, страницы — React-компоненты в `web/src/pages/*`, единый дизайн «сканер» (токены в `index.css`, компоненты `Reveal`/`UI`/`Nav`/`Footer`). Новости и анонсы обновляет CI (python-шаг в `.github/workflows/deploy.yml`).

**Tech Stack:** Vite + React 19 + TS + Tailwind 4 + Framer Motion; GitHub Actions → Pages.

## Global Constraints

- Контент — ТОЛЬКО из `tppkuban/raw/kuban-tpprf-ru-content-2026-07-23.md` (vault) и проверенных URL из `kuban-nav.txt`; ничего не выдумывать, при нехватке факта — ссылка на основной сайт

- Верификация каждой задачи: `npm run build` (0 ошибок) + просмотр страницы в превью `tppit-web`
